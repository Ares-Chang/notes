# FQ

争做遵纪守法好公民，一切行动听指挥！一切以国家利益为重！

  • Chrome 插件（简单）

    • iGG谷歌访问助手

    扩展程序自行百度使用方法。

    传送门：iGuge Helper (opens new window)，iGG 助手极简插件版 (opens new window)，Chrome 官方版 (opens new window)

    • 谷歌访问助手

    谷歌访问助手可自行百度，注意，百度安装的可能包含病毒或只能使用 12 小时，时间到后需要绑定 2345网址链接。

    类似问题可先暂时使用，然后访问 Chrome 商店 (opens new window) 安装正版插件，后卸载。

【推荐】插件易用，安装注册即可使用，无限流量，只是时常失灵，只可用于学术上网，不能观看视频。

  • 蓝灯（简单）

安装即用，简单粗暴，分为免费版及专业版，免费版有固定额度高速流量，如果用尽仍然可以使用，只不过会被限速。也可购买专业版，自行斟酌。

传送门: 蓝灯官方论坛 (opens new window)，Windows7及以上版本 (opens new window)，Mac OS X 10.11 及以上 (opens new window) 其他版本可点击论坛查看。

  • XX-Net（有难度）

基于 Google 免费的云服务搭建，But，搭建前你得先 FQ，然后才能下载软件。所以请自行选择。

传送门: 官方中文文档 (opens new window)，基于 Chrome 浏览器的搭建步骤 (opens new window)

  • 自己搭建 ssr (有难度)

最好还是自己搭建一个 ss/ssr 服务器来进行转载，如果想要追求稳定并有一定挑战精神，可以参考 walter201230 大神的教程文档 (opens new window)

以上方法请按需选择，本文参考自 walter201230 GitHub (opens new window)，感谢分享。为防消失，存档处理。
